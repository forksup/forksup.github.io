CSE 310 Storm Analysis

This is the 2nd project for my CSE 310 Data Structures class completed in the fall semester of 2019. The project required use to create a graph representation of a climatoloical data set. We utilized a storm data set provided by the National Weather Service (NWS). A full description of the project can be found here.

About the project

This project focuses around a few important data structures: Hash tables, arrays, linked lists, max-heaps, and binary search trees. We were tasked with parsing large lists of storm data, then building data structures around the given data. Hash tables were used to reduce search times, as well as heaps and binary search trees.

Data Structures Utilized
  • Hash Table
  • Max Heap
  • Binary Tree

After loading all the data into varying data structures, queries can be executed. There are several queries such as find max fatality and range which utilize data structures most effecient for their purpose. For example, range uses a binary search stree. Since this command requires easy access to a range of values with a low and high this data structure is ideal. Another example is find max fatality which builds a max-heap since it needs access to the storms with the most fatalities. A description of all of these commands can be found in p3 of the project requirements

Objectives
  • Use a large data set to demonstrate comprehension of varying data structures to reduce time complexity for searches and other data analysis.
  • Calculate Chain length as well as count for the storm linked lists
  • Calculate the load factor
