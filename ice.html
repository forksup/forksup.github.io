CSE 310 Seaice Analysis

This is the 3rd project for my CSE 310 Data Structures class completed in the fall semester of 2019. The project required us to create a graph representation of a climatological data set. We utilized a sea ice concentration anomaly data set from years 1979 to 2005. A full description of the project can be found here.

About the project

This project is a deep dive into graph theory. We explore a "small-world" graph which is characterized by a high degree of local clustering and a small number of long-range connections. Meaning nodes have a high degree of connectivity to neighboring nodes but the opposite for distant nodes.

This script creates a correlation-based graph of 3.3m data points by building an adjacency list and matrix. Three different graphs are created for three different alpha values to compare Pearson correlation coefficients. Furthermore, Floyd Marshall's shortest path, mean clustering coefficient, and degree distributions are calculated to compare with a random graph. Data conclusions can be found in the project state.

Objectives

  • Read a large (~3.3 million) data set of a 63x63x832
  • Analyze the data to create an adjacency matrix and adjacency list by calculating the Pearson value and comparing them to three alpha values .9, .95, and .99
  • Computer shortest path, clustering coefficient, and degree distribution for all three values.
  • Find whether or not the sea ice is a Small-world network.
