BookWorms YouTube

As part of my UGTA position for CSE 310 Data Structures and Algorithms under Violet Syrotiuk, I have created a YouTube channel to facilitate my communication with students. This year is entirely remote, which produces challenges and opportunities. On this channel, I help students learn C++ developer tools. As well as review test material.

About my YouTube channel

As a student, I have noticed my peers lack certain fundamental computer science skills that have not been taught sufficiently in their time at ASU. I have had to show colleagues how to use debuggers and breakpoints, which is crucial to the development and should be covered in coursework. Many students resort to using print statements, which are helpful but less powerful than watch statements used with breakpoints. To help with this problem I have created debugging videos to explain these concepts to my students.

Remote Deployment

Another challenging aspect of CSE 310 is the programming language, C++. This is a low level and technical language. An issue students frequently run into is variations between C++ compilers. There are many versions of the C++ compiler GDB which can produce various results with the same script. To remediate this, I created a remote deployment procedure to use in class. This technique uses a remote compiler on ASU's servers. Users do not need any compiler installed on their local machine, and use an IDE with linting all utilizing the remote compiler. This is useful as students do not need to worry about variations between their compiler and the compiler used to grade their work.
