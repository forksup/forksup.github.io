CSE 412 Book Playlist Project

In CSE 412: Database management, my group was tasked with selecting an existing data set and importing it into our own Postgres SQL database to be used as part of a web app. We decided to create an online book playlist project utilizing a large book database provided by Goodreads. Our project can be found here.

About the Project

In this project, we created a web app where users can search for a library and save books in their favorite lists. Once a favorites list is created the app will recommend a new book. The technologies used:

  • PostgresSQL
  • Bootstrap
  • React
  • Node

To make this process more simple we opted for a passwordless login page. Visitors may enter the site and select create to generate a new user id or enter in any other user id to view their stored data. This is for display purposes mostly.

Our web app allows users to search our database and save books by adding them to their favorites list:

After a user adds items to their favorites list, they can select the recommend button which will analyze their favorite books and recommend a new one. Our app analyzes users with similar favorite books and recommends new ones from their lists.
